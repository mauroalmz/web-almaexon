Preguntas frecuentes

Todo lo que necesitás saber antes de conectarte

¿Es seguro dar el ID de mi PC?

Sí, porque vos tenés que aceptar manualmente la conexión cada vez. Nadie puede acceder sin tu permiso expreso. Además, el ID cambia en cada ejecución si reiniciás el programa.

¿Qué problemas pueden resolver por remoto?

Lentitud extrema, virus, errores de sistema, configuración de correo, impresoras, redes WiFi, instalación de software, backups y mucho más. Si es un problema físico (hardware), te lo indicamos y coordinamos visita.

¿Tiene costo el diagnóstico remoto?

El diagnóstico inicial es sin cargo para nuevos clientes. Si ya sos parte de algún plan de mantenimiento, el soporte remoto está incluido. Para reparaciones puntuales, te pasamos presupuesto antes de empezar.
